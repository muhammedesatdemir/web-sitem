Faz 03 · Derin Teknoloji

BERTurk — Türkçe Duygu Analizi (Bitirme Projesi)

Otel yorumları · çok-modelli karşılaştırma · metrik bütünlüğü · XAI

Ar-Ge · Tamamlandı

Türkçe otel yorumları üzerinde üç sınıflı (negatif / nötr / pozitif) duygu analizi ve açıklanabilir AI (XAI) üzerine, iki kişilik tamamlanmış lisans bitirme projesi. Odak noktası yalnızca yüksek doğruluk değil; veri sızıntısını engelleyen dürüst bir değerlendirme çerçevesi kurmaktı: sentetik (yapay zekâ üretimi) örnekler yalnızca eğitim setinde tutuldu, test seti %100 gerçek kullanıcı yorumuyla izole edildi. Sentetik veri teste karıştığında metriğin yapay biçimde şiştiği (metric leakage) deneysel olarak gösterildi.

  • Dört Transformer mimarisinin eş protokolde karşılaştırılması (BERTurk · ELECTRA-tr · mBERT · XLM-RoBERTa)
  • ai_only_in_train izolasyonu + üç yöntemli sızıntı denetimi (exact-match · normalized · Jaccard)
  • Sınıf dengesizliği için Weighted Cross Entropy + karşıtlık bağlaçlı 751 insan-yazımı nötr yorum
  • İstatistiksel doğrulama (multi-seed · McNemar · 5-fold CV) + LIME / SHAP ile dilbilimsel açıklanabilirlik
  • FastAPI + React tabanlı uçtan uca demo (tek yorum tahmini, LIME açıklaması, metrik panosu)
BERTurk PyTorch · Hugging Face XAI · LIME / SHAP FastAPI React
