# Muhammed Esat Demir - Kişisel Portföy Web Sitesi

Bu proje, Yazılım Mühendisliği öğrencisi Muhammed Esat Demir'in kişisel portföyünü, deneyimlerini, yeteneklerini ve iletişim bilgilerini sergilemek amacıyla oluşturulmuş tek sayfalık (one-page) bir web sitesidir. Proje, Bootstrap 3 tabanlı açık kaynaklı **Browny** şablonu kullanılarak geliştirilmiştir.

## 🔍 Proje Hakkında

Bu web sitesi aşağıdaki bölümlerden oluşur:

- **Hakkımda:** Kısa biyografi ve iletişim bilgileri
- **Eğitim:** Eğitim geçmişi ve alınan dersler
- **Yetenekler:** Teknik ve kişisel beceriler
- **Deneyim:** Kulüp faaliyetleri ve staj geçmişi
- **Profil:** LinkedIn ve GitHub bağlantıları
- **Portföy:** Tamamlanan projeler
- **İletişim:** Ziyaretçilerin mesaj gönderebileceği bir iletişim formu

## 🛠️ Kullanılan Teknolojiler

- HTML5 & CSS3
- JavaScript & jQuery
- Bootstrap 3
- Font Awesome
- Animate.css
- Owl Carousel
- Bootsnav
